⌨️ Copier / Coller au clavier
1. Maintenez Ctrl enfoncé, puis appuyez sur C pour Copier après sélection du texte.
2. Cliquez dans un champ de saisie.
3. Maintenez Ctrl enfoncé, puis appuyez sur V pour Coller.
💡 Menu contextuel : clic droit de souris après sélection du texte.
